Archivos:
- index.html
- style.css
- script.js
- assets/ (coloca aquí musica.mp3 y fotos si después quieres agregarlas)

Publicación en GitHub Pages:
1. Entra a github.com e inicia sesión.
2. Crea un repositorio nuevo, por ejemplo: xv-jazmin.
3. Sube index.html, style.css, script.js y la carpeta assets.
4. Ve a Settings > Pages.
5. En Branch selecciona main y /root.
6. Guarda. GitHub te dará un enlace parecido a: https://tuusuario.github.io/xv-jazmin/

Para cambiar datos:
- Fecha/cuenta regresiva: script.js, línea eventDate.
- WhatsApp: index.html, enlace wa.me.
- Direcciones/horarios: index.html, sección events.
- Colores: style.css, sección :root.
